# FAEQA - Design Verification Assignment Collection

## Overview

This repository contains 15 hands-on assignments completed as part of my Design Verification training. The tasks cover a broad range of topics in the RISC-V ecosystem, including assembly programming, privilege modes, exceptions, interrupts, Python automation, RTL design, functional verification using Cocotb, and test generation using AAPG.

Each task is organized into its own folder and includes the source files, supporting files, and documentation describing the objective, implementation, verification method, and learning outcomes.
